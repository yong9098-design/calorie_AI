🔑 API 엔드포인트
================================================================================

    GET /api/config
    ├─ Purpose: Supabase URL/Key 조회 (환경 변수 프록시)
    ├─ Auth: 필요 없음 (공개)
    └─ Response: {supabaseUrl, supabaseAnonKey}

    POST /api/analyze
    ├─ Purpose: Gemini 음식 분석
    ├─ Auth: Optional (사용자 모델 선택용)
    ├─ Body: {image_base64: string, model?: 'flash'|'pro'}
    └─ Response: {food_name, calories, protein, carbs, fat, quantity, components}

    GET /api/food-search
    ├─ Purpose: 공공 식품 DB 검색 프록시
    ├─ Query: ?q=<음식명>&page=1&rows=10
    └─ Response: [{id, name, calories, protein, carbs, fat, ...}]
