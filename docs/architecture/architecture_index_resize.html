음식 분석 데이터 흐름

사진 촬영부터 DB 저장까지의 전체 과정

  • 1. 사용자가 사진 촬영 또는 갤러리 선택
  • 2. Base64 인코딩 후 API 전송
  • 3. Gemini API로 분석 (음식명, 칼로리, 영양소)
  • 4. Supabase meal_logs 테이블에 저장
  • 5. 결과를 사용자에게 표시 (수정 가능)
