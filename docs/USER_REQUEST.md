# 칼로리 계산 앱 요청

사용자가 자신의 일일 칼로리 필요량(TDEE)을 계산하고, 섭취한 음식의 칼로리를 기록하며, 목표 달성 현황을 추적할 수 있는 웹 앱을 원합니다.

## 요구사항 (비정제)

1. 기본 정보 입력: 나이, 성별, 키, 체중, 활동 수준
2. TDEE 자동 계산
3. 음식 추가 시 AI 기반 칼로리 인식
4. 일일 섭취 칼로리 합계 표시
5. 목표 대비 진행 상황 시각화
6. 회원가입/로그인 기능
7. 데이터 클라우드 저장

## 추가 요청사항

- 모바일 반응형 디자인
- 깔끔하고 직관적인 UI
- 어두운 모드 지원
- 월별 통계 리포트
